This project is a responsive Premium News Webpage designed using HTML, CSS, and JavaScript that fetches and displays live news data from the GNews Public API. It provides users with the latest headlines in a clean, modern, and visually attractive interface.

The webpage includes a Black and Gold premium theme for a luxury and professional appearance. It also supports Dark/Light mode, allowing users to switch themes easily. The layout is fully responsive, so it works smoothly on desktops, tablets, and mobile devices.

Users can browse news by selecting different categories such as General, Technology, Business, Sports, and Health. A country filter allows headlines to be viewed from different regions like the US, India, and the UK. The search bar helps users quickly find news on specific topics.

The website also includes a Breaking News Ticker, which automatically scrolls the latest headlines across the top of the page. News articles are displayed in attractive cards containing an image, title, source name, publish date, short description, and a Read Article button that opens the full story.

Modern CSS styling techniques such as gradients, shadows, glassmorphism, hover animations, rounded corners, and clean typography are used to create an aesthetic user experience.

This project demonstrates API integration, dynamic content loading, responsive design, and modern front-end development skills. It is ideal for students, beginners, and portfolio projects who want to showcase their ability to build real-world web applications using public APIs.
